About

Campus Life Planner

What is this?

Campus Life Planner is a task management app built specifically for university students. It helps you track assignments, events, and personal tasks, all in one place, with no sign-up required.

Why I built it

This app was built as a summative assignment for the Frontend Web Development module at African Leadership University (ALU), Kigali, Rwanda. The goal was to build a fully functional, accessible, and responsive web app using only vanilla HTML, CSS, and JavaScript.
